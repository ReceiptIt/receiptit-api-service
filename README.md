# ReceiptIt API Service

API Service

## Build and Run
#### 1. Clone Repository
    git clone https://github.com/ReceiptIt/receiptit-api-service.git
#### 2. Go to service folder
    cd receiptit-api-service
#### 3. Install npm package
    npm install
#### 4. Database migrate
    npm run db:migration
#### 5. Database seed
    npm run db:seeder
#### 6. Run server
    npm start


## Tests
#### Unit Tests
    npm run unit
#### Integration Tests
    npm run it
#### Lint
    npm run jshint
